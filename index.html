SWAPI - Star Wars API

Introduction
SWAPI provides you an information query function for Star Wars films.

Dependency

Installation
  1. Server Installation
    Input the command in the terminal. 
    go get -u github.com/S-Vanguard/StarWar_Server
  2. Run Server
    In the directory "/StarWar_Server", input the command followed to start the server. The Server starts once the terminal shows "Server started" 
    go run main.go
  3. Client Installation
    Download the files of StarWar_Client.
  4. Run Client
    Open the browser, get to the website 'localhost:8080'. Then you can get to the index of our SWAPI.
  5. Enjoy using SWAPI !

Usage
Input url like before and the result is returned in JSON format. If you are a member of our application, you can also enjoy the amazing parsing JSON function.
